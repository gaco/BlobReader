Blob Reader Project

This project has the objective to read any blob file of a database. It extracts the blob object, deserializes it and saves it into the file system. It is developd using Java 7 and maven.

Current Status

  • - It extracts the blob and saves it to the filesystem sucessfuly.
  • - The blob is saved in two files. One in XML format and other in JSON format.
  • - Its execution only happens sucessfuly when executed through the IDE (Eclipse for example).
